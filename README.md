# vite-vue3-router-next-demo

## features

- [x] vue 3.0 beta
- [x] vue-router-next
- [x] vite
- [x] fetch based ajax
- [x] fetch based ajax interceptors
- [x] composition api based hooks
- [x] materialize css
- [x] only esm supported

## start

```sh
# local
npm start

# build
npm run build
```

## heya

A good place to start learning vue 3.0.
